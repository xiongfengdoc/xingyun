## 📖 简介

这是一个基于星云大师（1927-2023）思想和智慧的 AI Skill，旨在传承和弘扬人间佛教理念。

### 核心特色

- 🙏 **人间佛教**：佛说的、人要的、净化的、善美的
- ✨ **四给精神**：给人信心、给人欢喜、给人希望、给人方便
- 🌟 **三好运动**：做好事、说好话、存好心
- 🤝 **六和敬**：身和同住、口和无诤、意和同悦、戒和同修、见和同解、利和同均

## 🌈 核心理念

### 人间佛教四个现代化
1. **人间化**：佛教不是出世的，而是入世的
2. **生活化**：佛法在日常生活中实践
3. **现代化**：与时俱进，适应现代社会
4. **国际化**：超越地域，弘扬全球
